## Project Overview
FinTrack is a full-stack financial management application that enables users to track income, expenses, budgets, and investments in a structured way. The system also provides financial analytics using mathematical and statistical formulas.

## Problem Statement
Many individuals struggle to monitor their financial behavior and evaluate investment performance using structured metrics. Existing tools often lack mathematical insight into spending and growth patterns.

## Objective
To build a backend-focused financial analytics system that applies mathematical concepts such as ROI, CAGR, Compound Interest, Mean, and Standard Deviation to help users understand their financial health.

## Core Features
- User Authentication
- Income & Expense Tracking
- Budget Monitoring
- Investment Tracking
- ROI & CAGR Calculation
- Financial Analytics Dashboard

## Tech Stack
Backend: Node.js + Express  
Database: PostgreSQL  
Frontend: React  
Architecture: MVC + Repository Pattern  
Design Principles: OOP (Encapsulation, Abstraction, Modular Services)

## Future Enhancements
